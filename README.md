Bocchi-meshi Bokumetsu Bento-bako
=================================

> You'll never eat alone anymore


Setup
-----

```sh
$ git clone git@github.com:creasty/b3_prototype.git && cd dely
$ mysql.server start  # if need
$ ./script/setup
```


Update
------

After git pull, run this command:

```sh
$ ./script/update
```
